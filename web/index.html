This web client demonstrates the JavaScript Rendering Manager Library for the Rust City Builder game. The library provides a modular, reusable interface for processing rendering commands sent from the Rust backend.

Features:

  • Modular JavaScript architecture with separate rendering and communication libraries
  • Canvas-based rendering with support for grids, sprites, and shapes
  • Command history and statistics tracking
  • Responsive design with real-time status updates
  • Error handling and connection management

Libraries Used:

  • rendering-manager.js - Core rendering functionality
  • web-client.js - Server communication and client management
